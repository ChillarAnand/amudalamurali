About

The main aim of the project is to provide simple access public domain telugu books.


Roadmap

1. Collect Resources

Gather various sites where public domain telugu books are available. Scrape those sites and build a database.

2. Clean Data

Collected books database should be cleaned. This includes removing duplicate books, providing telugu tranlations for missing titles, adding metadata e.t.c.

3. OCR

A good Telugu OCR software is needed. All books should be OCRed and generate epub/mobi books.
